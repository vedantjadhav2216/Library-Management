# 📚 Library Management System (C Language)

A simple console-based **Library Management System** developed using **C Programming Language** and **File Handling**. This project allows administrators to manage book records and customers to view and search available books through a login-based system.

---

## 🚀 Features

### 👨‍💼 Admin Module
- Secure Admin Login
- Add New Book
- Display All Books
- Search Book by ID
- Update Book Details
- Delete Book Record

### 👤 Customer Module
- Customer Login
- View Available Books
- Search Book by ID
